# AutoMind и Smart Garage Content

Проект **AutoMind** — B2C SaaS сервис рационального подбора автомобилей на основе объективных данных. Пользователь вводит свои параметры (бюджет, тип кузова, расход, надёжность и т.д.) и получает персонализированный шорт‑лист из 3–5 автомобилей. Для каждого варианта приводятся ключевые характеристики, сильные и слабые стороны, а также ссылки на независимые обзоры (YouTube, Drom, Отзовик, тематические форумы). Базовая версия шорт‑листа доступна бесплатно. Расширенные отчёты (глубокий анализ, прогноз затрат, расчёт владения) предоставляются по подписке.

**Smart Garage Content** — медиа‑проект, поддерживающий AutoMind. Мы создаём полезный контент на YouTube, Telegram и Pinterest: инженерные разборы типичных поломок, инструкции по чтению отзывов без рекламы, сравнение узлов и агрегатов разных марок. Каждый материал содержит нативные ссылки на AutoMind, позволяя пользователям получить персонализированную подборку.

## Структура репозитория

```
├── app.py                  # Flask‑приложение (веб‑сервер)
├── requirements.txt        # список зависимостей Python

│   ├── __init__.py
│   ├── shortlist.py        # генерация шорт‑листа автомобилей по параметрам

│   └── report.py           # формирование расширенного отчёта
├── data/
│   └── cars.csv            # пример набора данных об автомобилях
├── templates/              # HTML‑шаблоны для веб‑интерфейса
│   ├── base.html
│   ├── index.html
│   ├── shortlist.html
│   └── report.html
├── static/
│   └── style.css           # стили для интерфейса
├── content/                # примеры материалов для Smart Garage
│   ├── youtube/
│   │   └── script1.md
```

## Запуск

1. Установите зависимости:

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

2. Запустите сервер:

   ```bash
   FLASK_APP=app.py flask run --reload
   ```

3. Откройте `http://localhost:5000` в браузере.

MVP‑версия использует статический набор данных (`data/cars.csv`) и заглушки в `aggregator.py`. В дальнейшем сюда можно подключить реальные парсеры, модель машинного обучения и интеграцию платёжной системы.
